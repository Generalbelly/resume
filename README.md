## 強み

**「作る」と「届ける」の両方を担える点**  
スタートアップで創業期から携わり、技術選定・設計・実装だけでなく、顧客ヒアリング・要件定義・リリース調整・サポートまで一気通貫で担当してきました。ビジネスインパクトを念頭に優先順位を判断しながら開発を進められます。

**初速と品質のバランス**  
「とりあえず動かす」スピードと、「スケール時に壊れない」設計の両立を意識しています。担当プロダクトでは接続広告アカウント数を 3,000 から 20,000 へスケールさせた経験から、初期設計の重要性を身をもって理解しています。

**AI ツールの実践的な活用と展開**  
生成 AI を使ったプロダクト開発を複数経験しており、Claude Code を業務に導入して非エンジニア職種（広告運用者）への展開・勉強会開催まで行いました。ツールとして使いこなすだけでなく、組織・チームへの浸透まで担える点が強みです。

創業メンバーとしてジョインし、BigQuery を最大限活用した[広告レポーティングツール](https://www.data-be.at/)を CTO の右腕としてゼロから開発。

チーム規模: エンジニア 8〜10 人、デザイナ 0〜1 人、PM/営業 1 人

- **接続広告アカウント数を 3,000 → 20,000 へスケール**
  - プロダクトのスケーラビリティ課題を解決し、広告レポート取得数を約 200,000 件/日まで拡大した。BigQuery のパーティション設計や非同期処理の最適化により実現した
- **広告レポート取得アプリケーションのサーバーレス化（アーキテクチャ刷新）**
  - 既存の GAE モノリスを Cloud Run + PubSub の非同期アーキテクチャに刷新し、スケーラビリティと運用コストの両面を改善した
  - 使用技術: Go, PubSub, Cloud Run, Datastore, Memorystore, Cloud Scheduler, BigQuery
- **データマート構築アプリケーションの開発**
  - 広告データと顧客独自データを紐付けるデータマート生成システムを設計・開発した
  - 使用技術: Go, PubSub, Cloud Run, Datastore, BigQuery
- **Web アプリケーションのフルスタック開発**
  - BI ツール用 SQL 生成・Excel レポート生成アプリをフルスタックで開発した
  - 使用技術: Javascript, Vue.js（Vuetify）, PHP（Laravel）, GCE
