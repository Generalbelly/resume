## Project Overview

Bilingual (Japanese/English) resume repository. Markdown files are the source of truth; PDFs are derived artifacts committed to Git.

- `README.md` вҖ” Japanese resume (иҒ·еӢҷзөҢжӯҙжӣё)
- `en/README.md` вҖ” English resume
- `README.pdf` вҖ” Japanese PDF (generated externally, e.g. browser print)
- `en/Resume.pdf` вҖ” English PDF (generated by `en/generate_pdf.py`)

## Generating the English PDF

```bash
python3 en/generate_pdf.py
```

Requires Python 3 with `reportlab` (`pip3 install reportlab`).

The script produces a styled two-column PDF with green accent bars, matching a professional resume template. When updating resume content, edit the data directly in `en/generate_pdf.py` (the `main()` function contains all content), then regenerate.

## Workflow

1. Edit markdown files for content changes (both `README.md` and `en/README.md` should stay in sync)
2. Update `en/generate_pdf.py` if the English PDF content needs to change
3. Run `python3 en/generate_pdf.py` to regenerate the English PDF
4. Commit both markdown and PDF files together
